# serpentile

六角形サーペンタイル(serpentile)のランダム敷詰めを対話的に探索するシミュレータです。街で見かけた舗装の写真——六角タイルの上を白い紐がうねる模様——が発端で、「この紐が囲む面積に法則はあるか?」という素朴な問いから始まった探究の成果物です。

**デモ:** https://tokuichi-1091.github.io/serpentile/ 

## タイルについて

正六角形の各辺の中点を3本の弦で結んだタイルを、60°刻みのランダムな向きで平面に敷き詰めます。6つの中点をすべて1回ずつ使うため、どの向きの組み合わせでも紐は必ず隣のタイルへ繋がります。

この種の六角形タイルは、wikipediaによるとKurt N. Van Ness の命名で serpentile と呼ばれています。正方形の先祖は Truchet タイル(S. Truchet, 1704 / C. S. Smith, 1987)として知られています。

6辺を3組にペアリングする方法は回転を除いて5クラスしかなく、本作のタイルは辺の隔たりが **{1, 2, 2}** の唯一「弦同士が交差する」クラスです。これはVan Nessが考案した表記法における分類021のタイルで、一意な回転位置を6つ持ちます。

## 機能

- **紐(ロープ)表現** — 縁取り付き二重線。交差部は [fa][bc] → [ab][ef] → [cd][de] → 六角形輪郭の重なり順で描画
- **角の丸め(表示のみ)** — 見た目は滑らかな曲線、面積計算は元の折れ線のまま
- **3つの表示モード** — 囲まれた領域を面積で塗り分け / ループごとに色分け / 紐のみ
- **タップで情報表示** — 領域の面積(量子化単位)、紐の線分数・符号付き面積
- **トーラス盤面** — 対辺を接続。開放端が消え、代わりに「トーラスに巻き付く非可縮ループ」が観測できる
- **編集モード** — ヘックスをタップして時計回りに60°回転。ループの合流・分裂を手で遊べる
- **シード入出力** — 同じシード+同じ設定で同じ盤面を再現
- **盤面コード** — 全タイルの向きを可逆シリアライズ。編集後の盤面もコピペで完全再現

### 盤面コード仕様

```
h1.<p|t><サイズ>.<データ>
```

`p` は平面窓(サイズ=範囲R)、`t` はトーラス(サイズ=一辺N)。データ部は各タイルの向き k ∈ {0..5} を2タイルずつ1文字(`36進 k₁×6+k₂`、英数字)に詰めたものです。

## この模様の数学

シミュレーションと証明で分かったことの要約です。辺長1、面積の単位 U = √3⁄16(六角形の1⁄24)とします。

**面積の量子化(証明済み)** — 紐が囲むすべての領域の面積は √3⁄16 の整数倍。紐の頂点全体がある市松格子に乗ることによります。

**ループ面積の法則(予想)** — 閉ループの符号付き面積は 3√3⁄16(六角形の1⁄8)の整数倍。数千ループで反例なし、証明は未完です。

**平均は六角形1枚(証明済み)** — オイラーの公式より、窓を無限に広げると領域の平均面積は六角形1枚分に収束します。

**紐はなかなか閉じない** — 有限の窓では約8割の線分が「窓内で閉じない紐」に属し、この割合は窓を4倍にしてもほぼ変わりません。すべての紐が確率1で閉じるのか、無限に伸びる紐が正の密度で存在するのかは、ランダムに向きを置いた鏡で粒子軌道を追う**ミラーモデル(Lorentz lattice gas)**の仲間として、未解決問題の圏内にあります。どちらのシナリオでも「1本の紐を追ったときの長さの期待値」は発散します。

**トーラス化すると** — 開放端は消えますが、消えた分は「トーラスを (a, b) 周回する非可縮ループ」に化けます。平面で開放紐だった約8割の線分は、トーラスでは巻き付きループに属します。

**一筆書きは常に可能(証明済み)** — どのサイズのトーラスでも、全タイルの紐を1本のループにする配置が存在します。鍵は有限チェックで済む合流補題:「1つのタイルの6つの切り口が外部でどう繋がれていても、6つの向きのどれかを選べばそこを通る紐はすべて1本に合流する」。任意の配置から、タイルを1個ずつ回すだけで、ループ数を一度も増やさずに一筆書きへ到達できます。

**ループ密度の最大は 2⁄3(証明済み)** — タイル数Tに対しループ総数は L ≤ 2T⁄3(N≥4のトーラス)。証明は課金論法で、(1) どのループも通過する長弦の本数は偶数(方向差の総和が一周で0になることから)、(2) 実現可能なループはすべて 2×(短弦数) + (長弦数) ≥ 6 を満たす、(3) 短弦に重み1⁄3、長弦に重み1⁄6を配ると盤面総重量が 2T⁄3 で各ループが重み1以上を担ぐ、の3段です。そして6の倍数のNでは等号を達成する周期6×6の「結晶」が存在します: 頂点三角形(短弦×3)+菱形4ループ(短2+長2)+首飾り(1タイルを囲む長弦×6)による完全パッキングです。

### ギャラリー: 最密配置

盤面コード欄に貼り付けると読み込めます(ループ色分けモード推奨)。

最小胞 N=6(24ループ = 2T⁄3):

```
h1.t6.dygfo0gdy0foygdo0f
```

N=18(216ループ):

```
h1.t18.dygdygdygfo0fo0fo0gdygdygdy0fo0fo0foygdygdygdo0fo0fo0fdygdygdygfo0fo0fo0gdygdygdy0fo0fo0foygdygdygdo0fo0fo0fdygdygdygfo0fo0fo0gdygdygdy0fo0fo0foygdygdygdo0fo0fo0f
```

作り方も単純で、向きの行モチーフ A = [2,1,5,4,2,4](偶数行)と B = [2,3,4,0,0,0](奇数行)を、2行ごとに2マス斜めにずらして敷くだけです。
